# Milestone 1

## Objective
Develop a secure Employee Wellness Management System with user authentication.

## Features
- User Registration
- Email OTP Verification
- Secure Login using JWT
- PostgreSQL (Neon DB) Integration
- Streamlit Web Interface

## Technologies Used
- Python
- Streamlit
- PostgreSQL (Neon DB)
- JWT Authentication
