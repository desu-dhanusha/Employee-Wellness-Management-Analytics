# Employee Wellness Management Analytics – Milestone 2

## Project Objective

This milestone focuses on building a Natural Language Processing (NLP) preprocessing pipeline for employee feedback analysis. The system accepts employee feedback in TXT or CSV format, preprocesses the text, detects the language, and prepares the data for sentiment and emotion analysis.

---

## Features

- Upload employee feedback (TXT/CSV)
- Language Detection
- Unicode Normalization
- Text Cleaning
- URL & Email Removal
- HTML Tag Removal
- Emoji Extraction
- Punctuation Removal
- Number Removal
- Lowercase Conversion
- Tokenization
- Stopword Removal
- Lemmatization
- Sentiment Analysis
- Emotion Detection
- Processing Statistics

## How to Run

1. Open the notebook in Google Colab.
2. Install all required libraries.
3. Run all notebook cells.
4. Launch the Streamlit application.
5. Upload a TXT or CSV file.
6. Click **Run NLP Analysis**.
7. View preprocessing, sentiment, and emotion analysis results.
